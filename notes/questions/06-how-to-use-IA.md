# How do you use AI beyond just writing code?

I use AI tools across different parts of my product engineering workflow, ***not only to write code faster***.

For example: 
- I use Linear MCP to get ticket context faster, understand acceptance criteria, and stay aligned with product priorities. 
- I use Figma MCP to review designs, inspect details, and translate UI decisions more accurately into implementation. 
- I also use PostHog MCP to set track events accross the platform 

On the engineering side, 

- I use AI code review workflows like Superpowers to get a second pass on my PRs before sharing them with the team. 
- I also use OpenSpec-style workflows, like Superpowers: Superpowers is a framework of AI skills that helps structure the development process. It supports a Spec-Driven Development approach by helping define the plan first, then guiding implementation and code review based on that plan.

- I use AI for refactoring, exploring UI and UX alternatives with Claude Design,
