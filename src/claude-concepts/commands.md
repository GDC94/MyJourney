# skills
Instalar npx skills add https://github.com/juliusbrussee/caveman --skill caveman

Sirve para optimizar el resultado el output token. Menos palabras, menos tokens.

  Un skill de comunicación ultra-comprimida. La idea: bajar el uso de tokens ~75% hablando "como cavernícola"
  PERO manteniendo toda la sustancia técnica intacta. Muere el relleno, no el contenido.

  Para usarlo, antes de empezar a trabajar, correr /caveman. Luego, colocar /stop-caveman
